# kepinsam.tech
Personal site of Cevin Samuel.

## Getting Started

First, run the development server:

```bash
git clone https://github.com/kepinsam/kepinsam.tech.git
```
```bash
cd kepinsam.tech
```
```bash
pnpm i
```
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
