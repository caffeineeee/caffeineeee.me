## Getting Started

First, run the development server:

```bash
git clone https://github.com/caffeineeee/caffeineeee.me.git
```
```bash
cd caffeineeee.me
```
```bash
pnpm i
```
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
