# kepinsam.gg
Cevin Samuel personal site.

## Getting Started

First, run the development server:

```bash
git clone https://github.com/kepinsam/kepinsam.gg.git
```
```bash
cd kepinsam.gg
```
```bash
pnpm i
```
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
